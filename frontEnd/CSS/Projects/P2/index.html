About the platform

Built for Football, Powered by Data

WC26 Predict is an independent fan-built platform that turns raw football statistics into readable forecasts. We combine recent squad form, tournament-stage pressure and expected-goals modelling to project outcomes for all 104 matches of the 2026 FIFA World Cup — the first edition to be hosted across three nations and expanded to 48 teams.

This is a concept demo: all fixtures, odds and analysis are illustrative projections generated for entertainment, not official FIFA data or betting advice.
